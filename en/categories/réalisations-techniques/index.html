Introduction to Kubernetes

Kubernetes is an open-source container management system. It was designed to automate the deployment, scaling, and management of container-based applications. Main concepts of Kubernetes Kubernetes is based on different concepts that together form a coherent system to carry any application. Node Cluster Container Pod Deployment Service Node A node is a physical or virtual machine that is part of a Kubernetes cluster and performs cluster tasks. Each node has an agent software called kubelet, which monitors the state of containers on the node and keeps them running.
