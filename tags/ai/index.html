How does AI work ?

AI, which stands for Artificial Intelligence, is a branch of computer science that aims to develop machines capable of performing tasks that typically require human intelligence, such as speech recognition, decision-making, and visual perception. At its core, AI involves the use of algorithms, which are a set of instructions that guide a computer’s actions. Machine Learning is a particular subfield of AI that uses algorithms to learn from data and improve performance on tasks over time.
